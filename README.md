# MIMHO Solana

**MIMHO – The Memecoin of the Future**

MIMHO Solana documents the Solana-side launch of the MIMHO ecosystem.

This repository exists to provide transparency around the Solana launch, token identity, official links, risk disclosures, metadata and cross-chain roadmap.

## Launch Approach

The initial Solana launch is planned through a standard public launchpad flow.

No custom Solana/Rust program is deployed for this first Solana launch.

This means the Solana launch is intentionally simple, public and community-driven.

## Important Separation

MIMHO Solana is separate from the deprecated BSC Presale V1 contracts.

The BSC/EVM V2 contracts are maintained separately under MIMHO Labs and are undergoing security hardening.
