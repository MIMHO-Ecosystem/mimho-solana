# MIMHO Solana Launch

## Overview

The initial MIMHO Solana launch is planned through a standard public launchpad flow.

The goal is to make the launch simple, transparent and community-driven.

## Launch Model

| Item | Status |
|---|---|
| Network | Solana |
| Launch type | Public launchpad / bonding curve flow |
| Custom Solana program | No |
| Private presale | No |
| Guaranteed migration | No |
| Guaranteed listing | No |

## Why No Custom Solana Program at Launch

The initial Solana launch does not require a custom Rust or Anchor program.

Using a standard public launchpad flow reduces unnecessary complexity and avoids introducing custom contract/program risk at the first launch stage.

The focus of this launch is:

- community formation
- transparent token identity
- public access
- simple participation
- official metadata
- clear risk disclosure

## Launchpad Flow

If the token is launched through a bonding curve platform, trading and migration behavior will follow the platform's public rules.

If the bonding curve completes, any migration will follow the launch platform's supported AMM path available at that time.

No specific AMM migration, exchange listing or liquidity outcome is guaranteed.

## Solana Launch Scope

The Solana launch is a meme/community launch.

It does not automatically activate:

- BSC staking
- BSC vesting
- BSC DAO
- BSC marketplace
- BSC bridge
- cross-chain claims
- guaranteed rewards

Those modules belong to the broader MIMHO ecosystem and will only be activated in phases after proper testing, deployment and public documentation.

## Post-Launch Updates

After launch, this repository should be updated with:

- official mint address
- official launchpad link
- official explorer link
- official DEX/AMM link if applicable
- final metadata link
- official website link

## User Verification

Users should verify all official links before interacting with any token or platform.

MIMHO will publish official links in this repository and through official communication channels.

## Risk Notice

Participating in a meme coin launch is risky.

There is no guarantee of profit, liquidity, migration, listing, long-term value or market demand.
