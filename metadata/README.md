# MIMHO Solana Metadata

## Overview

This folder contains metadata preparation files for the MIMHO Solana launch.

The final metadata may depend on the launch platform requirements.

## Planned Metadata File

The planned metadata file is:

token-metadata.json

## Important Notice

The metadata file may contain placeholder values before launch.

Final values should be updated before or immediately after launch, including:

- final image URL
- official website
- final description
- official social links if supported

## Metadata Principles

MIMHO metadata should be:

- accurate
- simple
- transparent
- consistent with official branding
- free of guaranteed profit claims
- free of guaranteed listing claims
- free of guaranteed migration claims

## Suggested Token Description

MIMHO – a Meme Coin do Futuro. A cross-chain meme ecosystem starting with a public Solana community launch.

## Post-Launch Update Checklist

After launch, update:

- mint address
- launchpad link
- explorer link
- DEX/AMM link if applicable
- official metadata link
- final token image URL
